# osu! AI (Reinforcement Learning Project)

This project is a small experimental AI that learns to play **osu!** using **reinforcement learning**.

The AI was trained in a custom Python environment that simulates osu!-style hit circles appearing on the screen. The model learns by receiving rewards for correct timing and penalties for mistakes. Circles spawn randomly so the AI learns general timing behavior rather than memorizing a fixed pattern.

The system observes the screen and decides when to click based on the circle timing, similar to how a player would react during gameplay.

This was a personal project created to explore machine learning, computer vision, and game interaction using Python.
